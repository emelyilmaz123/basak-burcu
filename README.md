# Başak Burcu ♍ — Astral Arşivler

Başak burcunu anlatan, retro CRT estetiğiyle tasarlanmış interaktif astroloji sayfası.

**Canlı Site:** [basak-burcu-git-main-emelyilmaz123s-projects.vercel.app](https://basak-burcu-git-main-emelyilmaz123s-projects.vercel.app)

## Özellikler

- Animasyonlu yıldız alanı ve kayan yıldızlar (Canvas API)
- Mouse parallax efekti
- Nebula arka plan blob'ları
- Altın shimmer ve glitch animasyonları
- Günlük yorum sistemi (tarih bazlı dönen içerik)
- Kariyer kartları hover flip efekti
- Aşk uyumu çubuk grafikleri
- Kişilik özellikleri, güçlü/zayıf yönler, ünlü Başaklar bölümleri

## Teknolojiler

- Next.js 16
- Tailwind CSS v4
- Framer Motion v12
- Canvas API

## Kurulum

```bash
npm install
npm run dev
```
